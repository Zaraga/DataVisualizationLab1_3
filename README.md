# Data Visualization Lab - Visual Studio Solution

## What is included
- `DataVisualizationLab.sln` - Visual Studio solution
- `DataVisualizationLab.pyproj` - Python project
- `main.py` - lab code that loads the Spotify dataset, makes charts, prints summaries, and finds matching songs
- `spotify-2023.csv` - dataset file
- `report_template.md` - short writeup for submission

## How to run in Visual Studio
1. Open `DataVisualizationLab.sln` in Visual Studio.
2. Make sure you have Python support installed in Visual Studio.
3. Install packages from `requirements.txt` if needed.
4. Run `main.py`.
5. The script will save charts into the `output` folder and print the results in the terminal.

## Expected results from this dataset
- Typical BPM range: 100 to 140
- Most common key: C#
- Most common mode: Major
- Typical danceability range: 57 to 78
- Typical energy range: 53 to 77
- Typical speechiness range: 4 to 11
- Typical acousticness range: 6 to 43
